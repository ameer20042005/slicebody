# Medical Image Viewer

تطبيق سطح مكتب لعرض الصور الطبية فقط، بدون أي مكوّنات ذكاء اصطناعي أو تقسيم آلي.

## المزايا

- تحميل ملفات `DICOM` و`NIfTI` و`NRRD` و`MHD`
- استعراض المقاطع `Axial / Sagittal / Coronal`
- عرض حجمي ثلاثي الأبعاد باستخدام `VTK`
- أدوات `window/level` والسطوع والتباين والعكس
- حجم تجريبي مدمج للتجربة السريعة

## التشغيل

1. أنشئ بيئة بايثون وافعلها.
2. ثبّت الاعتمادات:

```bash
pip install -r requirements.txt
```

3. شغّل التطبيق:

```bash
python main.py
```

## بنية المشروع

- `main.py`: نقطة التشغيل
- `gui/`: واجهات العرض ثنائية وثلاثية الأبعاد والنافذة الرئيسية
- `medical/`: تحميل الصور الطبية والمعالجة الأساسية
- `visualization/`: أدوات العرض الحجمي عبر `VTK`
- `utils/`: إعدادات التطبيق والثيم

## ملاحظات

- تمت إزالة كل ملفات ومنطق الذكاء الاصطناعي من المشروع.
- التطبيق الآن مخصص للعرض والاستكشاف البصري فقط.
